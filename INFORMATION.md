# Instructions to setup the Project

-   run npm install
    To download all the libraries needed.

-   npx playwright install
    To setup all the instructions needed to run Playwright.

# Instructrions for run the project

-   npm run dev
    the project will run in the http://localhost:3000

### for run the functional test for the UI
